# Blinkendes Herz @tutorial
## @fullscreen
Lass dein microbit ein Herz zeigen!
Ziehe einen ``||basic:zeige LEDs||`` Block n den `||basic:dauerhaft||` Block.
Klicke auf die Kästchen, um ein Herzmuster zu zeichnen.
## 
Jetzt soll das Herz blinken! Füge einen zweiten `||basic:zeige LEDs||` Block unter dem ersten ein.
Lass diesen Block leer – so werden alle LEDs ausgeschaltet.
## 
Zu schnell? Füge zwischen den beiden `||basic:zeige LEDs||` Blöcken
einen `||basic:pausiere (ms)||` Block ein.
Stelle die Zeit auf 500 Millisekunden (= eine halbe Sekunde).
## 
Super! Klicke auf Herunterladen, um das Programm auf deinen microbit zu übertragen.
Das Herz blinkt jetzt auf deinem micro:bit! 💛
